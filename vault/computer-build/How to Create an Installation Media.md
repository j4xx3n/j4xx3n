## Install Rufus

1. If Rufus is not installed on a windows device you can download it from IT/Shared (rsd2-fs2.morgridge.net/IT/Shared/Software/Rufus-4.5.exe).
2. Open Rufus.

## Create Installation Media

1. Under Device select the USB drive you want to install the ISO on.
2. Under Boot Selection click the select button and find the ISO image for the OS you want to install in IT/Shared (r2d2-fs2.morgridge.net/IT/Shared/ISO Images).
3. Click start. (**Warning this will wipe the USB drive!)**         
4. Once the status bar is complete click "Close". 

![[Pasted image 20260328182650.png]]

5. You can now remove and boot into the USB to install the OS image.
