Kubuntu is a free, open-source operating system that combines Ubuntu's rock-solid base with the KDE Plasma desktop. Here's everything you need to get it installed.

## Required Materials

- A USB drive (8 GB or larger)
- At least 25 GB of free disk space
- An internet connection (recommended during install


## Instructions
1. Go to **kubuntu.org/getkubuntu** and download the latest Kubuntu ISO file. Save it to your Downloads folder.
2. Create a bootable drive with the Kubuntu ISO via Rufus. [How to Create an Installation Media](https://j4xx3n.github.io/j4xx3n/categories/computer-build/)
3. Boot from the USB
4. **Install Kubuntu** — jump straight to the installer
5. The installer will ask about a couple of setup options:
	- **Install third-party software** — recommended; adds things like media codecs
	- **Download updates while installing** — recommended if you're connected to the internet
6. **Use entire disk**
	- Wipes the drive and installs Kubuntu as the only OS. ⚠️ All existing data will be erased.
7. Pick your region and city. This sets your system clock automatically.
8. Select your keyboard language and layout. There are many regional variants available.
9.  **Create Your User Account**
	- **Name** (your display name)
	- **Username** (used to log in)
	- **Password** (also used for installing software and system updates — keep it somewhere safe)
	- **Computer name** (hostname)
10. **Install**
	- Click **Install Now** and confirm. The installation typically takes 10–20 minutes. When it finishes, you'll be prompted to restart.
