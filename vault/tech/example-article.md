This is an example tech article. Replace this file with your own notes.

## How this works

Your Obsidian vault folder structure maps directly to blog categories:

- `vault/tech/` → category: **Tech**
- `vault/personal/` → category: **Personal**
- Each `.md` file becomes an article

## Tips

- You don't need front matter — the script fills it in automatically
- The article title defaults to the filename with dashes replaced by spaces
- The date defaults to when you first committed the file to git
- Add tags in front matter if you want them: `tags: ["tag1", "tag2"]`
